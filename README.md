# 🎓 Student Home Tutor System

A web application designed to connect students with instructors for home tutoring. The platform simplifies tutor discovery, communication, and user management through a secure and user-friendly interface.

## ✨ Features

- 🔐 Role-based authentication
- 👨‍🎓 Student dashboard
- 👨‍🏫 Instructor dashboard
- 📚 Tutor management
- 👥 User management
- 📊 Interactive dashboards
- 📱 Responsive design

## 🛠️ Tech Stack

### Frontend
- Next.js

### Backend
- REST API

### Database
- MySQL

## 🚀 Getting Started

### Prerequisites

- Node.js
- npm
- MySQL

### Installation

```bash
git clone https://github.com/SenaEnana/dts-full.git

cd dts-full

npm install

npm run dev
```
